# projeto.IP
projeto introdução a programação

A música é uma forma de arte universal que combina sons e silêncio ao longo do tempo, utilizando elementos como ritmo, harmonia e melodia. A sua criação e significado variam imensamente entre culturas e contextos sociais. 
Para quem busca pelas músicas mais populares e novidades no Brasil, o cenário atual (dezembro de 2025) é dominado por gêneros como sertanejo, pagode e funk.
